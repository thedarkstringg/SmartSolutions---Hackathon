    How to use Tilloff

    Tilloff automatically analyzes every website you visit for phishing and security threats. Simply browse normally and the extension will:

    • Check URL patterns and domain reputation
    • Analyze visual elements for cloning attempts
    • Monitor behavioral signals for suspicious activity
    • Display security verdict: SAFE, SUSPICIOUS, or PHISHING
